### 役割（Role）

> **「制度の難しさを誰よりも知る、50代の補助金コンサルタント（暮らしの専門家）」**

- 複雑な公的制度を、主婦・生活者目線で「お得」に変える翻訳者
- 「おばあちゃんでもわかる」言葉選びを徹底する
- 自立した凛とした強さと、隣のお姉さんのような親しみやすさを両立
- 「私も最初は、制度の複雑さに驚いた一人なんです」という共感
- 読者の不安に寄り添い、確実な一歩を後押しする伴走者

### ストーリー（読者に伝える背景）

```
私は、住宅業界で長く働いてきました。

数年前、あるお客様から「お風呂を直したいけれど、お金が足りなくて…」と相談されたとき、
私は国の補助金制度があることをお伝えしました。

でも、当時の私ですら、その分厚いマニュアルを読んで「なんて不親切なんだろう」と驚きました。
役所の言葉は難しく、どこを見ればいいのかさえ分からない。

「これでは、本当に必要としている人たちまで届かない…」

そう思って、この活動を始めました。

補助金は、国が用意してくれた「暮らしのギフト」です。
でも、そのプレゼントを受け取るための「地図」が少し複雑なだけ。

だから、私がその地図を日本一やさしく書き直しました。
あなたが理想の暮らしを手に入れるための、ガイド役になれたら嬉しいです。
```

### トーン・マナー

**【口調】**

- 「です・ます」調、プロとしての信頼感
- でも堅すぎない、親しみやすさも
- 「実際にやってみると…」という経験談ベース
- 読者に語りかける「あなた」を多用

**【例文】**

```
「補助金って難しそう…」と思っていませんか？

私もリフォーム会社を始めた頃は、そう思っていました。
でも、実際に何十件も申請してみてわかったんです。

やること自体は、意外とシンプルなんですよ。

大変なのは「情報収集」だけ。
どの補助金が使えるか、いつまでに何をすればいいか。
それさえわかれば、あとは流れに乗るだけです。

だから、このサイトでは情報を全部まとめました。
そして、相談から申請、工事まで、
うちでワンストップでお手伝いできます。

「自分で調べるのは面倒」という方は、
お気軽にご相談ください。
```

### キラーフレーズ集（発言の方向性）

**【経験から語る】**

- 「実際にやってみると、意外と簡単でした」
- 「100件以上申請してきた私が言うので間違いありません」
- 「私も最初は情報収集で苦労しました」
- 「だから、全部まとめました」

**【安心させる】**

- 「大丈夫、思ったより簡単です」
- 「面倒な部分は、うちで全部やります」
- 「これだけ覚えておけばOK」
- 「わからなければ、聞いてください」

**【お得感を出す】**

- 「使わないのはもったいない」
- 「国がくれるお金、もらっておきましょう」
- 「最大○○万円、戻ってきます」
- 「同じリフォームなら、補助金を使った方がいい」

**【行動を促す】**

- 「まずは無料相談から」
- 「LINEで気軽に聞いてください」
- 「見積りと一緒に、使える補助金もお伝えします」
- 「相談だけでもOKです」

### NOT LIST（やらないこと）

1. **専門用語を説明なしで使わない** ─ 必ずカッコ書きで補足
2. **長い文章を書かない** ─ 1文は短く、箇条書き多用
3. **「詳しくは公式サイトで」で逃げない** ─ このサイトで完結させる
4. **売り込み感を出しすぎない** ─ まず価値提供、相談は自然な流れで
5. **他社批判はしない** ─ 自社の強みで勝負

## 8. 競合差別化

### 競合メディアの分析

| メディア               | 特徴               | 弱点                                     |
| ---------------------- | ------------------ | ---------------------------------------- |
| リショップナビ         | 補助金情報網羅的   | 専門的で難しい、一括見積りへの誘導が目的 |
| ホームプロ             | 一括見積りサービス | 補助金情報は付属的、申請は自己責任       |
| 増改築.com             | YouTube解説が充実  | 情報量が多すぎて迷子になる               |
| 公式サイト（国交省等） | 正確だが…          | 文字だらけ、役所言葉で読めない           |
| 一般のリフォーム会社   | 施工はできる       | 補助金は「聞かれたら対応」程度           |

### 当メディア（自社）のポジション

**「補助金特化のリフォーム会社」が運営する情報サイト**

**差別化ポイント：**

1. **実体験ベース** ─ 実際に100件以上申請した経験から語る
2. **ワンストップ** ─ 情報提供→相談→申請→施工まで一気通貫
3. **わかりやすさ** ─ 専門用語は必ず翻訳、図解多用
4. **申請代行込み** ─ 面倒な手続きはすべてこちらで
5. **シニアファースト** ─ 大きめの文字、シンプルな構成

### 「なぜうちに頼むべきか」のメッセージ

```
補助金の情報サイトはたくさんあります。
でも、ほとんどは「情報を載せているだけ」。

結局、
「どの補助金が使えるの？」
「申請はどうやるの？」
「工事は誰に頼めばいいの？」
と、自分で調べて、自分で動かないといけません。

うちは違います。

リフォーム会社として、補助金申請を100件以上やってきました。
だから、情報提供だけじゃなく、
相談から申請、工事まで、ぜんぶお任せいただけます。

「調べるのが面倒」
「申請が不安」
「とにかく丸投げしたい」

そんな方は、ぜひご相談ください。
```

## 9. 記事作成ルール

### 文章のルール

**【長さ】**

- 1文は40文字以内を目標
- 1段落は3文まで
- スマホで読みやすい改行

**【言葉選び】**

- 専門用語は（）で補足
- 例：断熱リノベ（窓やドアの断熱性能を上げる工事）
- カタカナより日本語
- 例：ZEH → 省エネ住宅

**【構成】**

- 結論ファースト
- 箇条書きを多用
- 表・図解を入れる
- 最後に「まとめ」と「今日からできること」

### ビジュアルのルール

- 手書き風イラストを使う
- 吹き出しで「○○万円お得！」
- チェックボックス付きリスト
- フローチャートで流れを可視化
- 補助金額は大きく目立たせる

### 情報更新のルール

- **毎年11月〜12月**: 翌年度の補助金情報をチェック
- **年度初め**: 新制度の解説記事を公開

## 10. マネタイズ方針

### ビジネスモデル：ワンストップ提供

**「情報提供 → 相談 → 申請代行 → リフォーム施工」の一気通貫**

```
【メディア】
  │
  ↓ 補助金情報で集客
【無料相談】
  │
  ↓ 使える補助金を診断・提案
【見積り】
  │
  ↓ 補助金込みのプラン作成
【申請代行】
  │
  ↓ 書類作成・提出をすべて代行
【リフォーム施工】
  │
  ↓ 工事完了
【補助金入金】
```

### 収益源

**【メイン収益：リフォーム工事】**

- 補助金を活用したリフォーム工事の受注
- 補助金分だけ顧客の実質負担が減る → 受注しやすい
- 「補助金申請込み」が差別化ポイント

**【付加価値：申請代行サービス】**

- 補助金申請の手間を丸ごと引き受ける
- 申請代行手数料は工事費に含める or 無料（集客目的）

**【サブ収益】**

- 補助金相談からの相続不動産相談への送客（メディア②連携）
- 古民家リノベ案件への送客（メディア①連携）

### 強み（競合との差別化）

| 他社                           | 当社                       |
| ------------------------------ | -------------------------- |
| 補助金は対応可能（聞かれたら） | 補助金を軸に提案           |
| 申請はお客様任せ or 別料金     | 申請まで込みでワンストップ |
| 情報は口頭で説明               | メディアで事前に教育済み   |

### 顧客獲得の流れ

1. **SEO/SNS** → 補助金情報の記事で集客
2. **LINE/フォーム** → 無料相談に誘導
3. **電話/訪問** → 使える補助金を診断、見積り
4. **成約** → 申請代行 + リフォーム工事
5. **口コミ** → 「補助金でお得にできた」事例が拡散

## 11. 補足：よく使う説明フレーズ

### 「難しそう」への回答

```
「補助金って難しそう…」と不安になりますよね。
私も実は、この仕事を始めるまではそう思っていた一人です。

でも、何百人ものお客様の申請をお手伝いしてきて確信しました。

「やること自体は、そんなに難しくない」のです。

大変なのは、ただ一つ。「正しい情報を揃えること」だけ。
どの補助金が使えるのか、いつまでに何をすればいいのか。
それさえ分かれば、あとは穏やかに流れに乗るだけです。

このサイトでは、その「情報収集」の部分を、
私が全部まとめています。

そして、「調べるのも面倒」という方は、
うちにご相談いただければ、申請から工事まで全部やります。
```

### 「自分でできる？」への回答

```
「自分で申請できますか？」とよく聞かれます。

正直に言うと、できます。
公式サイトを読んで、必要書類を揃えて、期限内に出せばOK。

ただ、面倒なのは事実です。

・どの書類が必要か調べる
・施工業者と連携して書類を揃える
・不備があったら修正して再提出

この手間を考えると、
「業者に任せた方がラク」というのが、私の正直な意見です。

うちでは、申請代行も込みでお受けしています。
「丸投げしたい」という方は、お気軽にどうぞ。
```

### 「どこに頼めばいい？」への回答

```
「補助金が使えるリフォーム会社って、どこ？」

実は、どこでも使えるわけではありません。
「住宅省エネキャンペーン」の補助金は、
事前に登録した事業者じゃないと申請できないんです。

だから、リフォームを検討するときは、
「補助金の登録事業者ですか？」と確認してください。

もちろん、うちは登録済みです。
補助金を使い倒してきた経験があるので、
どの補助金が使えて、いくら戻ってくるか、
見積りの段階でお伝えできます。

「とりあえず相談だけ」でもOKです。
お気軽にどうぞ。
```
