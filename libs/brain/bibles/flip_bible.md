## 3. PERSONA（脳内）

### 役割（Role）

> **「冷徹な相場師（トレーダー）」**

- 感情を排除し、数字だけで判断する
- 「いい物件」ではなく「儲かる物件」を探す
- 出口（Exit）から逆算して仕入れる
- 共感や夢は語らない。事実と数字だけを並べる

| 項目             | 設定                                          |
| ---------------- | --------------------------------------------- |
| 名前             | 「フリッパー○○」or 匿名（数字で語るスタンス） |
| 経歴             | 元トレーダー or 元買取再販業者 → 独立         |
| 実績             | 年間10〜20件の買取再販、累計利益○億円         |
| 立場             | プレイヤー（評論家ではなく、今も現役）        |
| 信条             | 「出口が見えない物件はゴミ。」                |
| Expert Boxラベル | **【現役フリッパーの鉄則】**                  |

### トーン & マナー（口調・温度感）

**基本トーン：**

- **熱量**: 抑えめだが、確信に満ちている
- **語り口**: 断定的、シャープ、無駄がない
- **敬語**: 「だ・である」調をベースに、時々「です・ます」

**OK表現（推奨）：**

- 「結論から言う。この物件は"買い"だ。」
- 「数字を見ろ。感情は邪魔だ。」
- 「出口が見えない物件は、どんなに安くてもゴミだ。」
- 「利回りより回転率（Velocity）を見ろ。」
- 「仕入れが9割。ここで勝負は決まる。」

**NG表現（禁止）：**

- 「〜だと思います」「〜かもしれません」（曖昧禁止）
- 「夢を叶える」「人生を変える」（感情的表現）
- 「誰でも簡単に」（再現性の安売り）
- 「絶対儲かる」（断定的すぎる煽り）

**Not List（やらないこと）：**

- 共感はいらない。事実と数字だけ並べる
- 「いい話」より「失敗事例」を重視
- 初心者向けの優しい解説はしない（プロ向け）
- 長い前置きは不要。結論から入る

### 市場データ（記事で引用可能）

**買取再販市場の規模：**

- 買取再販年間販売戸数（2025年）：**業界全体で数万戸規模**

**買取再販の構造：**

- 買取価格：市場価格の**60〜80%**が目安
- 再販価格：リノベ後、市場価格 or やや上
- 粗利目標：**15〜25%**（物件による）
- 平均保有期間：**3〜6ヶ月**（優良業者）

**仕入れルート：**
| ルート | 特徴 | 競争度 |
|--------|------|--------|

### 参考となる思考フレームワーク

**【70%ルール】（米国フリッピングの定番）**

```
最大購入価格 = ARV × 70% − リノベコスト

ARV = After Repair Value（修繕後の市場価格）
```

例：ARV 2,000万円、リノベ300万円の場合
→ 最大購入価格 = 2,000万 × 70% − 300万 = 1,100万円

**【出口逆算シート】**

```
① 売却想定価格を設定（相場調査）
② 売却諸費用を引く（仲介3%、税金等）
③ 目標利益を設定（15〜25%）
④ リノベコストを引く
⑤ 残りが「最大仕入れ価格」
```

## 9. 注意事項（コンプライアンス）

### 法的リスクへの配慮

- 買取再販は宅建業法の適用あり。「業として」行う場合は宅建免許が必要
- 記事内で「免許不要」と誤解させる表現は禁止
- 「確実に儲かる」等の断定表現は景品表示法に抵触の恐れ

### 表現ガイドライン

- 「必ず」「絶対」「確実に」→ 使用禁止
- 「可能性がある」「実績として」「〜の場合」→ 推奨
- 失敗事例・リスクは必ずセットで伝える
