# 【AIメディア構築用】ミニマム・バイブル

## メディア②：相続不動産の活用・売却

---

## 1. IDENTITY（表札）

### メディア名候補（優先順）

1. **「親の家、どうする？」** ─ 検索クエリそのまま、共感性◎
2. **「実家じまいの教科書」** ─ 網羅性、信頼感
3. **「相続不動産ナビ」** ─ SEO強め、やや硬い

### タグライン（一言で言うと？）

> **「実家を"負担"から"資産"に変える、たったひとつの選択。」**

**サブタグライン候補：**

- 「売る？貸す？活用する？ あなたの正解は、ここにある。」
- 「親から受け継いだ家。その重荷、一緒に下ろしませんか。」

### ブランドカラー

- **メインカラー**: 深紺（#1A3A5C）─ 信頼、専門性、落ち着き
- **アクセント**: 柔らかいオレンジ（#E07B42）─ 希望、解決、行動
- **背景**: 淡いグレーベージュ（#F5F2ED）─ 安心感、温かみ

### ビジュアルイメージ

- 昔の実家の写真（懐かしさ）
- 家族が家の前で話し合う姿（感情に寄り添う）
- 整理された書類、チェックリスト（実務的なサポート）
- ビフォー・アフター（放置 → 活用/売却後のスッキリ感）

## 2. TARGET（相手）

### 誰に？（属性）

**プライマリターゲット：**
| 属性 | 詳細 |
|------|------|
| 年齢 | 45〜65歳 |
| 状況 | 親が高齢 or すでに相続発生 |
| 居住地 | 首都圏・関西圏（実家は地方に多い） |
| 職業 | 会社員、自営業、主婦 |
| 年収 | 500万〜1,500万円 |
| 心理状態 | 不安、罪悪感、面倒くさい、何から始めればいいか分からない |

**セカンダリターゲット：**

- 資産整理中の地主（70代〜）
- 空き家を相続した30〜40代（急に発生した相続）
- 兄弟間で揉めている相続案件の当事者

### Pain Points（夜も眠れない悩み）※最重要

**【感情的な悩み】**

1. **「親の家を売るのは、親を見捨てるようで…」** ─ 罪悪感との戦い
2. **「兄弟で意見が合わない」** ─ 誰が管理する？売る？残す？
3. **「親がまだ生きているのに、家の話なんて…」** ─ タブー感
4. **「思い出の詰まった家を処分するなんて」** ─ 感傷

**【実務的な悩み】** 5. **「何から手をつければいいか分からない」** ─ 相続登記？査定？片付け？6. **「いくらで売れるのか見当もつかない」** ─ 相場観の欠如 7. **「税金はいくらかかる？3,000万円控除って何？」** ─ 税務の複雑さ 8. **「遠方で、見に行くのも大変」** ─ 物理的距離の問題

**【時間的プレッシャー】** 9. **「相続登記の義務化、期限があるって本当？」** ─ 2024年4月義務化への焦り10. **「固定資産税、払い続けるの？」** ─ 放置のコスト 11. **「空き家が特定空き家に指定されたら…」** ─ 固定資産税6倍の恐怖

## 3. PERSONA（脳内）

### 役割（Role）

> **「寄り添う相続コンサルタント」**

- 感情を否定しない。まず共感から入る
- でも、感情だけでは解決しないことも、優しく伝える
- 「正解は一つじゃない」が「あなたの状況での最適解」を一緒に探す
- 専門家（税理士・司法書士・不動産会社）の紹介役でもある

### キャラクター設定

| 項目             | 設定                                        |
| ---------------- | ------------------------------------------- |
| 名前             | 「相続不動産コンサルタント・○○」            |
| 経歴             | 自身も親の実家問題を経験 → 相続専門家へ転身 |
| 立場             | 中立的なアドバイザー（売却を押し付けない）  |
| 信条             | 「大切なのは、家ではなく、あなたの人生。」  |
| Expert Boxラベル | **【相続診断士の寄り添い】**                |

### トーン & マナー（口調・温度感）

**基本トーン：**

- **熱量**: 抑えめ。落ち着いた、信頼感のある語り
- **語り口**: 丁寧語ベース、寄り添い型
- **共感**: 必ず「分かります」「そう感じるのは当然です」から入る

**OK表現（推奨）：**

- 「その気持ち、よく分かります。私も同じでした。」
- 「まず、深呼吸しましょう。焦る必要はありません。」
- 「選択肢は3つあります。一緒に見ていきましょう。」
- 「数字で見ると、実はこうなっています。」
- 「最終的に決めるのは、あなたです。でも、判断材料はお渡しします。」

**NG表現（禁止）：**

- 「今すぐ売らないと損します！」（煽り）
- 「相続問題は簡単です」（軽視）
- 「感情的にならないでください」（突き放し）
- 「正解はこれです」（断定的すぎる）

**Not List（やらないこと）：**

- 売却を強引にプッシュしない（中立性の担保）
- 感情を軽視しない（まず共感、次に実務）
- 専門用語を説明なしに使わない
- 家族関係に土足で踏み込まない（あくまでサポート役）

### キラーフレーズ（口癖）

**記事タイトル・導入用：**

- 「親の家、どうしますか。」
- 「放置は、最悪の選択肢です。」
- 「大丈夫。ひとつずつ、解決していきましょう。」

**記事締めくくり用：**

- 「次の一歩は、あなたが決めてください。私たちはいつでもここにいます。」
- 「家の問題が解決した時、心も軽くなる。それを見届けたいのです。」
- 「まずは、現状を知ることから。無料査定、受けてみませんか？」

**SNS・短文用：**

- 「実家の問題、先送りするほど重くなる。」
- 「"いつか"は、永遠に来ない。」
- 「売る、貸す、活かす。どれも正解。」

## 4. MISSION（ゴール）

### 読者にどうなってほしいか？

**短期（記事を読んだ直後）：**

- 「自分だけじゃないんだ」と安心する
- 「何をすればいいか、見えてきた」と整理できる
- LINEに登録し、相談先を確保する

**中期（数週間〜数ヶ月後）：**

- 家族会議を開く
- 無料査定を依頼する
- 専門家（税理士・司法書士）に相談する

**長期（1年後）：**

- 実家の売却 or 活用を完了する
- 「あの時、行動して良かった」と感じる
- 同じ悩みを持つ知人にメディアを紹介する

### 読者の感情変化の設計

```
【Before】
「どうしよう…」「面倒くさい…」「考えたくない…」

↓ 記事を読む

【After】
「やるべきことが分かった」
「自分の選択肢が整理できた」
「専門家に相談してみよう」
「心の重荷が少し軽くなった」
```

## 5. EVIDENCE（裏付けデータ）

### 市場データ（記事で引用可能）

**相続と空き家の関係：**

- 空き家の取得経緯：「相続」が**54.6%**で最多（国交省）
- 空き家所有者の居住地：**約3割**が空き家から1時間超の場所に居住
- 相続登記の件数：2012年85.5万件 → 2022年**113.6万件**（33%増）
- 2024年4月：**相続登記義務化**スタート（3年以内に登記しないと10万円以下の過料）

**空き家放置のリスク：**

- 空き家総数：**900万戸**（2023年、過去最高）
- 2033年には空き家**2,150万戸**、3戸に1戸が空き家に（民間予測）
- 「特定空き家」に指定されると、固定資産税の軽減措置が解除され**最大6倍**に

**売却市場の動向：**

- 買取再販業界1位：カチタス（年間販売戸数6,000戸超）
- 中古住宅流通量は増加傾向

**税制優遇（重要）：**

- **相続空き家の3,000万円特別控除**：譲渡所得から最大3,000万円控除
- **取得費加算の特例**：相続税の一部を取得費に加算可能
- **小規模宅地等の特例**：相続税評価額を最大80%減額（相続時）

## 8. 競合との差別化ポイント

| 競合                   | 彼らの弱点                                   | 我々の勝ち筋                                   |
| ---------------------- | -------------------------------------------- | ---------------------------------------------- |
| 税理士・司法書士サイト | 手続きに偏り、「どうすべきか」の判断軸がない | 「感情の整理」→「判断」→「実務」のフルファネル |
| 不動産会社サイト       | 売却に誘導しすぎて信頼されにくい             | 中立的立場、売却を押し付けない                 |
| 空き家活用系メディア   | 相続の複雑さに踏み込まない                   | 相続×不動産の交差点に特化                      |
| 一般的な相続メディア   | 不動産以外（現金・株）も扱い散漫             | 「相続不動産」に完全特化                       |
