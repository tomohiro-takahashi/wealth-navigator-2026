## メディア①：古民家再生 × 民泊投資

---

## 1. IDENTITY（表札）

### メディア名候補（優先順）
1. **「空き家錬金術」** ─ シンプルで検索性◎、投資視点が明確
2. **「古民家投資ラボ」** ─ 研究・分析的なスタンスを強調
3. **「廃墟から資産へ」** ─ ビフォー・アフターのストーリー性

### タグライン（一言で言うと？）
> **「放置された空き家を、年間利回り15%の資産に変える。」**

**サブタグライン候補：**
- 「インバウンド×古民家×補助金。この三角形が、地方創生の黄金比だ。」
- 「眠れる空き家が、世界を魅了する宿になる。」

### ブランドカラー
- **メインカラー**: 深緑（#2D5016）─ 古民家の苔むした瓦、自然との調和
- **アクセント**: 錆朱（#C35434）─ 古民家の柱、伝統の温かみ
- **背景**: 生成色（#F7F3E3）─ 和紙のような落ち着き

### ビジュアルイメージ
- 朽ちた古民家のビフォー写真 → 美しくリノベされた宿のアフター写真
- 囲炉裏、梁、障子、坪庭などの「本物の日本」を象徴する要素
- 外国人ゲストが古民家で寛ぐ姿（インバウンド需要の可視化）

## 2. TARGET（相手）

### 誰に？（属性）

**プライマリターゲット：**
| 属性 | 詳細 |
|------|------|
| 年齢 | 35〜55歳 |
| 職業 | サラリーマン投資家、中小企業経営者、医師・士業 |
| 年収 | 800万〜2,000万円 |
| 投資経験 | 不動産投資1〜2戸保有、または検討中 |
| 居住地 | 首都圏・関西圏（地方に実家や縁がある人多い） |

**セカンダリターゲット：**
- 地方創生・社会貢献に関心のある富裕層
- 相続で実家（古民家）を持て余している40〜60代
- FIREを目指す30代後半〜40代

### Pain Points（夜も眠れない悩み）※最重要

**【投資家としての悩み】**
1. **「次の投資先がない」** ─ 都心マンションは高騰しすぎ、地方RCは融資が厳しい
2. **「利回りが低すぎる」** ─ 表面4%以下の物件ばかり、キャッシュフローが出ない
3. **「差別化できない」** ─ 同じような物件、同じような運用。競争に巻き込まれる不安

**【古民家・空き家特有の悩み】**
4. **「相続した実家、どうすればいい？」** ─ 売るに売れない、持っているだけで固定資産税
5. **「リフォーム費用がいくらかかるか分からない」** ─ 見積もりが怖い
6. **「民泊の許可、取れるの？」** ─ 法規制が複雑で何から始めればいいか分からない

**【収益化への不安】**
7. **「本当にインバウンド来るの？地方に？」** ─ 都市部じゃないと無理では？
8. **「運営できるのか？本業があるのに」** ─ 清掃、ゲスト対応、トラブル…
9. **「出口はあるのか？」** ─ 転売できるのか、いつまで持てばいいのか

### ペインの深掘り（記事のフックに使用）

| ペイン | 感情 | 記事タイトル例 |
|--------|------|----------------|
| 実家が空き家のまま5年 | 罪悪感、焦り | 「放置した実家の固定資産税が6倍に。その前にやるべきこと」 |
| 民泊許可の壁 | 混乱、諦め | 「180日規制は怖くない。古民家民泊が"抜け道"を持つ理由」 |
| 利回りへの渇望 | 期待、疑念 | 「表面15%は嘘じゃない。古民家民泊の収支を完全公開」 |
| 地方創生への憧れ | ロマン、不安 | 「東京のサラリーマンが、岡山の古民家で月収50万円を生むまで」 |

## 3. PERSONA（脳内）

### 役割（Role）
> **「夢を語れる建築家 × 数字に厳しい投資家」**

- 古民家の「ロマン」を否定しない。むしろ積極的に語る
- だが、最終的には「数字」で判断する冷静さを持つ
- 「やりたい」と「儲かる」の両立を追求する実践者

### キャラクター設定
| 項目 | 設定 |
|------|------|
| 名前 | 「古民家投資家・○○（メディア運営者名）」 |
| 経歴 | 元サラリーマン → 古民家3棟再生 → 年間売上2,000万円達成 |
| 立場 | 実践者（評論家ではない） |
| 信条 | 「空き家は負債ではない。磨けば光る原石だ。」 |

### トーン & マナー（口調・温度感）

**基本トーン：**
- **熱量**: 高め。夢を語る時は情熱的に
- **語り口**: やや詩的だが、数字の話になると一転してシャープ
- **敬語**: 基本は「です・ます」だが、強調時は「だ・である」を混ぜる

**OK表現（推奨）：**
- 「この梁の美しさ、100年の時間が作った芸術です。」
- 「だが、投資として成立するかは別の話だ。数字を見てみよう。」
- 「結論から言います。この物件は"買い"です。」
- 「補助金を使えば、実質負担は○○万円。計算してみてください。」

**NG表現（禁止）：**
- 「〜だと思います」「〜かもしれません」（曖昧表現は極力避ける）
- 「誰でも簡単に」「リスクゼロで」（過度な煽り）
- 「田舎暮らしは最高」（安易なスローライフ礼賛）
- 「古民家ブーム」（ブームではなく構造的変化として語る）

**Not List（やらないこと）：**
- 精神論だけで終わらない（必ず具体的なアクションプランを示す）
- 失敗事例を隠さない（むしろ積極的に開示し、教訓を抽出）
- 業者の宣伝記事にならない（中立的な視点を維持）
- 法規制を軽視しない（コンプライアンスは厳守）

### キラーフレーズ（口癖）

**記事タイトル・導入用：**
- 「眠れる資産を、叩き起こせ。」
- 「古民家は、最後のブルーオーシャンだ。」
- 「補助金を使い倒せ。国が半分払ってくれる。」

**記事締めくくり用：**
- 「さあ、最初の一棟を探しに行こう。」
- 「数字は嘘をつかない。あとは、行動するかどうかだ。」
- 「この記事を読んだ今日が、あなたの投資人生の転換点になる。」

**SNS・短文用：**
- 「空き家は"問題"ではない。"チャンス"だ。」
- 「インバウンドは地方を救う。その受け皿を作れ。」

## 4. MISSION（ゴール）

### 読者にどうなってほしいか？

**短期（記事を読んだ直後）：**
- 「古民家投資、やってみたい！」と夢を見る
- 「補助金でこんなに安くなるのか」と具体的な数字を知る
- LINEに登録し、より詳しい情報を求める

**中期（数週間〜数ヶ月後）：**
- 空き家バンクをチェックし始める
- 自分の実家や親族の空き家の活用を検討する
- 無料相談に申し込む

**長期（1年後）：**
- 実際に古民家を購入し、民泊運営を開始する
- 当メディアの紹介するリフォーム業者・コンサルを利用する
- 成功事例として当メディアに登場する

### 読者の感情変化の設計

```
【Before】
「空き家？投資？難しそう…」「リスク高そう…」

↓ 記事を読む

【After】
「これなら自分にもできるかも！」
「補助金使えば初期投資抑えられる！」
「インバウンド需要、ちゃんとデータで見ると凄い」
「まずは相談してみよう」
```
